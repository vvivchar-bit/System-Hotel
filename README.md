\# Програмна система «Готель»



Навчальний прототип програмної системи для пошуку та бронювання номерів у готелі.



\## Опис проєкту



Програмна система «Готель» призначена для автоматизації основних процесів роботи готелю.  

У першій версії реалізовано простий прототип, який дозволяє переглядати список номерів, виконувати пошук, фільтрувати номери за статусом та створювати бронювання.



\## Функціональні можливості



\- перегляд списку номерів;

\- відображення типу, ціни та статусу номера;

\- пошук номерів за типом;

\- фільтрація номерів за статусом;

\- бронювання вільного номера;

\- введення імені та номера телефону клієнта;

\- автоматична зміна статусу номера на «Заброньований»;

\- перегляд списку створених бронювань.
