# Варіанти використання програмної системи «Готель»

Документ містить опис акторів та варіантів використання навчальної програмної системи «Готель».

## 1. Актори системи

| Актор | Опис |
|---|---|
| Клієнт | Користувач, який переглядає номери, виконує пошук і фільтрацію, бронює номер, скасовує бронювання та може оплачувати бронювання. |
| Адміністратор | Працівник готелю, який переглядає бронювання, контролює їх виконання та змінює статуси номерів. |
| Менеджер | Працівник готелю, який відповідає за керування номерами, їх описами, цінами, типами та статусами. |
| Керівник готелю | Користувач, який переглядає статистику роботи готелю та формує звіти. |
| Система оплати | Зовнішня система, яка забезпечує оплату бронювання. |
| Email-сервіс | Зовнішній сервіс, який використовується для надсилання підтвердження бронювання клієнту. |

## 2. Варіанти використання

| Основний актор | Варіант використання | Опис |
|---|---|---|
| Клієнт | Перегляд номерів | Клієнт переглядає список номерів готелю. |
| Клієнт | Пошук номера | Клієнт шукає номер за потрібним типом або параметром. |
| Клієнт | Фільтрація номерів | Клієнт фільтрує номери за статусом. |
| Клієнт | Бронювання номера | Клієнт створює бронювання вільного номера. |
| Клієнт | Скасування бронювання | Клієнт скасовує створене бронювання. |
| Клієнт | Оплата бронювання клієнтом | Клієнт ініціює оплату бронювання. |
| Адміністратор | Перегляд бронювань | Адміністратор переглядає список створених бронювань. |
| Адміністратор | Зміна статусу номера | Адміністратор змінює поточний статус номера. |
| Менеджер | Керування номерами | Менеджер керує інформацією про номери готелю. |
| Керівник готелю | Перегляд статистики | Керівник переглядає статистику роботи готелю. |
| Керівник готелю | Формування звіту | Керівник формує звіт про діяльність готелю. |
| Система оплати | Оплата бронювання | Зовнішня система обробляє оплату бронювання. |
| Email-сервіс | Надсилання підтвердження | Email-сервіс надсилає клієнту підтвердження бронювання. |
